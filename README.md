# Game of Life
Free Code Camp Project. react + sass + enzyme + webpack + hot reloading

## Installation
1. git clone https://github.com/Luiko/Game-of-Life
2. cd Game-of-Life
3. npm install
4. npm run build
5. npm start

## Development
* npm run dev
* npm run hot
* npm test
* node server.test.js

### optional
* add your own .env file with custom port
